# Bright-coffee-shop-sales-analytics

Bright coffee shop has recently appointed a new CEO whose mission is to increase revenue and sales. 

To do this, he need to understand the current state of the business. 

As a Junior data analyst for the company, I have been tasked with extracting insights from the data to assist the new CEO to make informed business decisions.

The business has given me the following objectives for the project to better understand the bussiness.

-Which products generate the most revenue
-What time of day the store performs best
-Sales trends across products and time intervals
-Recommendations for improving sales performance

In order to meet these objectives, the project has been broken down into 4 steps
1. Project planning
-Architecture diagram to show the overviw of the project from the raw data to actionable insights.
-A flowchart was drawn on Miro to show the plan for the project.
This flowchart hightlight the main objectives of the project, the columns that will be used to address these objectives, the SQL queries as well as the type of graphs that will be used to visualize.
-A project gantt chart was devised to show the timeline for the project

3. Data processing
- Data bricks will be used to process the data.
  
4. Data analysis and visualization in Excel
-Use pivot tables, graphs and charts for visualization.

4.Project presentation
-Summarise the findings in a powerpoint presentation.
-Give recommendations based on the insights.
